### Comment by hejny on 11/29/2024, 3:02:48 PM

Both are different levels of abstraction.

OpenAI library is for direct use of OpenAI API.

Promptbook engine and the book language is for a higher level of abstraction. It define pipelines that are independent of the underlying library, LLM model, or even LLM provider.
