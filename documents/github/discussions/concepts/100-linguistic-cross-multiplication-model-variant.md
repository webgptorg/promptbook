# Linguistic Cross-multiplication

Performs text transformation, cross-multiplication, extrapolation, interpolation according to text patterns

```
King -> Queen
Man -> ____
```

```
World -> Europe -> ____ -> Bavaria
```

```
Small -> Smaller -> ____
```

It is emulated by other models like [completion](https://github.com/webgptorg/promptbook/discussions/96) or [chat](https://github.com/webgptorg/promptbook/discussions/97) model or [transformed through vector space](https://github.com/webgptorg/promptbook/discussions/98).
