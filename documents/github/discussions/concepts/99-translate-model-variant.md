# Translate model

The translation model translates from one language to another or from one style to another, e.g. informal -> formal.

It can be also emulated by other models like [completion](https://github.com/webgptorg/promptbook/discussions/96) or [chat](https://github.com/webgptorg/promptbook/discussions/97) model or [transformed through vector space](https://github.com/webgptorg/promptbook/discussions/98).

-   Google translate
