# Core Concepts

Fundamental ideas and building blocks of the Promptbook ecosystem.

-   [**Agent**](./agent.md) - The modern abstraction of an AI entity.
-   [**Pipeline**](./pipeline.md) - The legacy version of Promptbook execution.
-   [**Promptbook**](./promptbook.md) - The project and the language itself.
-   [**Book File**](./book-file.md) - The file format used to define Agents and Pipelines.
-   [**Collection**](./collection.md) - A grouping of multiple Agents or Pipelines.
