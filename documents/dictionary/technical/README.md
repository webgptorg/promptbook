# Technical Terms

Underlying technologies, implementation details, and technical concepts of the Promptbook project.

-   [**LLM (Large Language Model)**](./llm.md) - The "brain" behind the AI.
-   [**Prompt**](./prompt.md) - The actual text sent to the LLM.
-   [**Model**](./model.md) - Specific AI models like GPT-4, Claude 3.5, etc.
-   [**Executor**](./executor.md) - The engine that runs Pipelines or Agents.
-   [**Scraping**](./scraping.md) - The process of extracting information from websites.
-   [**Script**](./script.md) - Custom code (Javascript/Python) used within a Pipeline.
-   [**Postprocessing**](./postprocessing.md) - Cleaning and transforming the LLM output.
-   [**Usage**](./usage.md) - Tracking and measuring resource consumption.
