# Harnesses section

The harnesses section shows that `ptbk coder` orchestrates existing coding agents.

## Supported harnesses

-   OpenAI Codex
-   Claude Code
-   GitHub Copilot
-   Cline
-   Opencode
-   Gemini

## Required message

`ptbk coder` does not replace these tools. It provides the prompt queue, context, verification feedback, progress visibility, and Git lifecycle around them.
