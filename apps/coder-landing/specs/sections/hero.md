# Hero section

The hero must answer what `ptbk coder` is before the visitor scrolls.

## Required content

-   Eyebrow: `CLI workflow for AI coding agents`
-   Product name visible in the header.
-   Headline: `ptbk coder`
-   Supporting tagline: `Prompt queues into shipped commits.`
-   Supporting copy must explain that `ptbk coder` wraps existing coding agents with prompt files, agent behavior, verification feedback, local progress UI, and Git bookkeeping.
-   Primary action link scrolls to the command walkthrough.
-   Secondary action link scrolls to workflow.

## Required visual

The hero visual shows:

-   A terminal frame with a `ptbk coder server` command.
-   Prompt loading and selected harness status.
-   A failed first verification followed by a second attempt.
-   A local board preview with ready, running, draft, and done columns.
