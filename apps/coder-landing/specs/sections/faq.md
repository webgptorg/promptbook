# FAQ section

The FAQ answers comparison and trust questions for developers.

## Required questions

1. Is `ptbk coder` another AI model?
2. Why use prompt files instead of one-off chat?
3. What happens when tests fail?
4. Does the server send code to a hosted ptbk service?

## Answer style

-   Be direct.
-   Avoid legal or security claims beyond current implementation.
-   Make clear that the selected harness controls which AI provider receives prompt context.
