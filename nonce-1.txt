Promptbook: Invisible AI Agents

Promptbook is a framework for creating persistent AI agents that turn a company's scattered knowledge into action, powered by the Agents Server (gallery.ptbk.io).

Key points:
- Agents are defined using simple "Books" - a human-readable language that is explicit, reliable, and portable
- The Agents Server is the core platform where agents live, remember context, collaborate in teams, and get things done
- The main challenge in business AI is managing context, providing rules/knowledge, and narrowing personality - Promptbook addresses this
- Agents can be given personas, goals, rules, knowledge sources (URLs/documents), and tools (email, browser, etc.)
- One-click deployment is available via Vercel with Supabase integration
- Supports OpenAI and other AI providers via API keys
- The project is open source on GitHub (webgptorg/promptbook) and published as an NPM package
