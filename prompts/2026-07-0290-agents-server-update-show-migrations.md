[✨🎎] The self-update of the Agents server should show information about Migration

-   On `/admin/update` of Agents server you can trigger the self-update of the server, it works propperly and shows lot of useful information about the update process, but it does not show information about the migrations that are applied during the update process.
-   Keep in mind the DRY _(don't repeat yourself)_ principle.
-   Do a proper analysis of the current functionality before you start implementing.
-   You are working with the [Agents Server](apps/agents-server)
