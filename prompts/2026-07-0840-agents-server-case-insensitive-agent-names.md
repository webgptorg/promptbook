[✨👅] Agent IDs should be case-insensitive

-   For example `https://lts3.ptbk.io/agents/doQMRg82izNfJa/chat`, `https://lts3.ptbk.io/agents/doqmrg82iznfja/chat` and `https://lts3.ptbk.io/agents/DOQMRG82IZNFJA/chat` should refer to the same agent.
-   Keep in mind the DRY _(don't repeat yourself)_ principle.
-   Do a proper analysis of the current functionality before you start implementing.
-   You are working with the [Agents Server](apps/agents-server)
-   Add the changes into the [changelog](changelog/_current-preversion.md)
